# PCAT

Photo sharing web app using Node.js, Express.js, Mongoose, EJS for CRUD operations. (Node.js starter app)

##### npm dependencies:
* express
* express-fileupload
* ejs
* method-override
* mongoose
* nodemailer *

## Installation:

1- Clone this project

2- Run the commands below:

```bash
npm install
```

```bash
npm start
```

3- App will be started at http://localhost:5000
